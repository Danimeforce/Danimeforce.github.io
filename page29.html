MODES

Modes are power-ups which are used in battle. There are different modes with many different uses. Some modes can be bought while some others cannot. Multiple modes cannot be used simultaneously(i.e only one mode can be used at a time).MODES can also been switched on and off so far you have the enough cp(switching on a mode takes one turn same as switching off a mode).
